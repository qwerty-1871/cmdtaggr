Run "help" to output this information
Run "tui" to use a guided interface

Otherwise, cmdtaggr is controlled using the following commands:

create [tag] [files] | Create a new tag
list | Print a list of existing tags
files [tag] | List the files in a tag
search [tag] [search query] | Search within a tag for files matching the input
add [tag] [files] | Add files to an existing tag
remove [tag] [files] | Remove files from a tag
delete [tag] | Delete a tag
